# Zerodha
A full-stack web application inspired by the Zerodha trading platform ,focusing on a clean UI and core trading functionalities.
## Features
* **Interactive Dashboard:**
* Real-time data visualization and user-friendly interface.
* **Responsive Design:** Optimized for both desktop and mobile viewing.
* **Backend Integration:**
* Handles data management and user authentication.

* ## Tech Stack
* **Frontend:** React.js ,Tailwind CSS
* **Backend:** Node.js ,Express.js
*  **Database:** MongoDB
*  ## What I Learned
*  How to structure a large-scale MERN stack Application
*  Implementing complex UI components to mimic a real-world fintech application
